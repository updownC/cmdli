## Introduction

Easy npm script runner!
Check all your package scripts and run them at once!

### Installation

---

```sh
npm i -D cmdli
```

### How to Use

---

1. run 'cmdli'

```sh
cmdli
```

2. read your scripts list and select one with keyboard interface.
